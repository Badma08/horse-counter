# Учет лошадей на ипподроме

Веб-приложение для автоматического учета лошадей на изображениях
с использованием предобученной нейронной сети YOLOv8.

## Функционал
- Загрузка изображений
- Детектирование и подсчет лошадей
- Визуализация результатов
- Сохранение истории запросов
- Генерация Excel-отчета

## Запуск проекта
```bash
pip install -r requirements.txt
python app.py
